# Leaflet For R

+ Sử dụng các package để làm việc với leaflet và dữ liệu GIS: Leaflet, Rgdal, ...

    + Chưa kết hợp với Shiny để trở thành Web
    + Dự định sẽ kết hợp với TVDI package trở thành service tính toán TVDI trên Web
